# Lesson 7 – JavaScript Basics (Arrays)

This project introduces arrays in JavaScript to store lists of items.

## Features
- Paragraph with id `displayText` updates dynamically
- Button `clickMeBtn` updates text
- Button `decisionBtn` uses `if/else`
- Button `repeatBtn` demonstrates loops
- Button `showArrayBtn` demonstrates arrays and displays a list

## Files included
- index.html – Webpage structure
- script.js – JavaScript logic
- README.md – Project explanation
